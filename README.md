## 📋 Description
Terba7 is a professional e-commerce platform built with
a modern and scalable architecture using MVC pattern and OOP principles.

## 🏗️ Tech Stack
- **Frontend**: Angular 19 + TypeScript
- **Backend**: Spring Boot 3 + Java 17
- **Database**: PostgreSQL 16
- **Security**: Spring Security + JWT
- **Container**: Docker + Docker Compose
- **Methodology**: Scrum + Kanban

## 🚀 Getting Started

### Prerequisites
- Java 17+
- Node.js 20+
- PostgreSQL 16+
- Docker

### Installation
```bash
# Clone the repository
git clone https://github.com/Firassayar/terba7-backend.git

# Navigate to project
cd terba7-backend

# Run the application
./mvnw spring-boot:run
```

## 📁 Project Structure
